# Moknah-ATS

## Article to Speech Functionality
The Article to Speech feature allows users to convert text articles into spoken audio format, enhancing accessibility and user experience.

## Per-Post Generation
This feature enables users to generate audio for each individual post, allowing for tailored audio content based on user preferences.

## API Key Request Process
To utilize the Article to Speech functionality, users must first register an account and request an API key. The process includes filling out a form and verifying the email address.

## Getting Started
1. Clone the repository
2. Install dependencies
3. Set up your API key in the configuration file
4. Start the application

For more information, please refer to the documentation.
