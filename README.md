# ATS (Article to Speech) WordPress Plugin

## Installation
1. Download the plugin ZIP file from the releases section.
2. Go to your WordPress admin area.
3. Navigate to `Plugins > Add New > Upload Plugin`.
4. Upload the downloaded ZIP file.
5. Activate the plugin after installation.

## Features
- Converts articles into spoken words.
- Supports multiple languages and voice styles.
- Easy integration with existing WordPress installations.
- User-friendly interface for managing settings.

## Setup
- Go to the `Settings > ATS` page in the WordPress dashboard to configure your plugin settings.
- Select your preferred voice and language options.
- Enter your API key to enable speech conversion.

## Usage
- After installation, you can use the ATS plugin to convert articles to speech by clicking the "Convert to Speech" button on the post editor screen.
- You can also set the plugin to automatically convert new articles.

## API Key Requirements
- An API key is required for the speech synthesis service. You can obtain it from [your API provider] and enter it in the plugin settings.
- Ensure that your API key has the necessary permissions for speech synthesis.
